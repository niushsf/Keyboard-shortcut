Keyboard Shortcuts

These are the shortcuts for the Mac programs we use in the Moblie Apps class.

Key Command Application
⌘-Space Spotlight Safari
⌘-S Save Safari
⌘-R Refresh Safari
⌘-T New Tab Safari/Windows
⌘-X Cut Safari/Windows
⌘-C Copy Safari/Windows
⌘-V Paste Safari/Windows
⌘-S Shift 4 Take picture
⌘-A Select all Document/Windows
⌘-W Close program Document/Window tab
⌘-N Opens new Blank document/window
⌘-H Hide View front app,hide other apps
⌘-O Open Select item or open file
⌘-P Print Document
⌘-Z Undo Undo apps
⌘-D Duplicate Selected files
⌘-E Eject Selected disk or volume
